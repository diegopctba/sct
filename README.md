# Contas a pagar API REST
API Sistema de contas a pagar.

# Instruções iniciais
1. Clonar o projeto
2. Construir o aplicativo com mavem (mvn clean install)
3. Rodar [docker-compose up -d]
4. Acessar http://localhost:8080/swagger-ui/index.html no navegador
5. Credenciais: 
    - usuário: username
    - senha: password
